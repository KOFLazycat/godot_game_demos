# DashDash

## 1. 游戏机制
- 类似幸存者游戏，带肉鸽元素
- 主要玩法就是冲刺，玩家可以上下左右移动，依靠冲刺瞬间移动，并给敌人造成伤害
- 玩家升级后会提升能力，能力包括冲刺距离、速度、伤害、路径伤害、终点伤害、冲刺CD、冲刺伤害范围、二段冲刺、三段冲刺、分身等等，伤害特效可参考火、冰、毒、雷等
- 会有BOSS
